# OA办公平台 - 独立项目详情页

这是一个独立的OA办公平台项目详情页网站项目。

## 安装依赖

```bash
npm install
```

## 运行开发服务器

```bash
npm run dev
```

项目将在 http://localhost:3001 启动

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```

## 项目结构

- `src/App.tsx` - 主应用组件，包含OA办公平台的完整详情页
- `src/index.css` - 全局样式和Tailwind CSS配置
- `source/` - 项目图片资源

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS

## 注意事项

- 此项目独立于主作品集网站
- 所有图片资源已包含在 `source/` 目录中
- 端口设置为 3001，避免与其他项目冲突
